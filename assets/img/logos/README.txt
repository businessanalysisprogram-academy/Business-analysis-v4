COMPANY LOGOS — drop your files here
=====================================

Save each company logo in THIS folder using these EXACT filenames
(all lowercase, .jpg, no spaces). The website already points to these names.

  amazon.jpg
  rak.jpg
  capgemini.jpg
  vertex-engineering.jpg

RULES
-----
1. Filename must match EXACTLY. The live server (Vercel) is case-sensitive,
   so "Amazon.JPG" will NOT work — use "amazon.jpg".

2. Do NOT double the extension. If your file is already an image, just rename
   it to "amazon.jpg" — do not type ".jpg" onto a name that already ends in
   .jpg/.jpeg/.png (that creates "amazon.jpg.jpeg", which fails to load).

3. JPGs have a white background (no transparency). That's fine — each logo
   sits inside a small white rounded chip, so a white-background logo blends in.
   A square-ish crop with a little padding looks best.

4. Until a file is added, the card shows a coloured letter (a / R / C / V)
   instead — that is the intended fallback, not a bug.
